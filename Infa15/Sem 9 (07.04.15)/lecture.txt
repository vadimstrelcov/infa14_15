Рассмотрим SMP
OpenMP - библиотека для SMP
#include <omp.h>
#pragma ... - инструкции, которые выполнются на этапе компиляции
#pragma omp parallel
{
}
OMP_NUM_THREADS - создать переменную окружения и записать кол-во потоков
omp_set_num_threads(int) - ф-ция, которая реализует вышесказанное
по умолчанию OMP_NUM_THREADS = числу ядер на компьютере
Для компиляции OpenMP необходим ключ -fopenmp
omp_get_thread_num() - id текущего потока 
omp_get_num_threads() - число потоков
omp_get_wtime() - кол-во секунд со дня основания linux
Расспараллеливание цикла
#pragma omp parallel for
for (i=0;i<N;i++) {
}
