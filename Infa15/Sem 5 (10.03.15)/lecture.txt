Иногда необходимо переопределить ф-цию в производном классе ф-цию из базового класса.

Виртуальная функция - ф-ция в базовом классе, для которой во всех производных класса возможно переопрееление.
virtual type func() - только в базовом классе
В производном классе type func();

Class1 <- virtual void func()
Class2:Class1 <- void func()
Class3:Class2 <- void func() переопределять можно во всех производных класса

Динамический полиморфизм
Class1* p1;
Class2* p2=new Class2;
Запись p1=p2 действительная.
p1->func(); //будет выполнена ф-ция func() из Class2

Абстрактный класс - базовый класс для других классов
Нельзя создавать объекты абстрактного класса, состоящего из виртульных функций.
