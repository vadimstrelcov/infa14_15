Перегрузка функции
int sum(int a, int b) {
	return a+b;
}
double sum(double a, double b) {
	return a+b;
}
int sum(int a) {
	return a+1;
}
3 разные функции с одним и тем же именем. Нельзя создавать 2 функции с одинаковыми именами и одинаковыми аргументами.
